📝 Todo List App (React + Vite)

A simple and responsive Todo List application built using React and Vite. This app allows users to add, delete, and manage tasks efficiently with a clean and minimal UI.

---

🚀 Features

- ➕ Add new todos
- ❌ Delete todos
- 🔍 Search todos by name
- 📋 Display all tasks dynamically
- ⚡ Fast performance using Vite
- 🎯 Simple and beginner-friendly code

---

🛠️ Tech Stack

- React – Frontend library
- Vite – Build tool for fast development
- CSS / Tailwind (optional) – Stylings
